# Movie Explorer

Movie Explorer is a web application that allows users to browse movies dynamically using [TMDB API](https://developer.themoviedb.org/). It features an interactive sidebar, hover effects with detailed movie info, and a responsive contact form with validation.

## Features

- **Movie Browsing:** Fetches movies from TMDB API (Now Playing, Popular, Top Rated, Trending, Upcoming)
- **Interactive UI:** Hover overlay displays full movie title, description, release date, and rating
- **Search Functionality:** Users can search for movies by title
- **Responsive Sidebar:** Hidden menu with category navigation and social icons
- **Contact Form:** Validated inputs including regex validation for email, phone, and password
- **Smooth Animations:** Image hover effects and sliding overlay transitions
- **Responsive Design:** Works across desktop and mobile devices

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla JS)
- Bootstrap 5
- FontAwesome icons
- TMDB API

## Getting Started

1. Clone the repository:  
   ```bash
   https://github.com/Habiba-Sherif11/movie-explorer.git
2. Navigate to the project directory
3. Open index.html in your preferred browser. You can simply double-click the file or use a live server extension (recommended for full functionality).
4. Start exploring movies:
   - Use the search bar to find movies by title.
   - Click the sidebar menu to filter by category (Now Playing, Popular, Top Rated, Trending, Upcoming).
   - Hover over any movie card to see the full description, release date, and rating.
